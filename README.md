# AI Scientist

This repository explores all of the PubMed publications of Boston Children's Hospital research (circa 20k files).

This repository contains PubMed dataset files for research and analysis.

## Dataset

The repository includes two PubMed dataset CSV files in the `data/` directory:
- `pubmed_data.csv` - Full dataset with 20,415 records (225 MB)
- `pubmed_data_2000.csv` - Subset with 2,000 records
